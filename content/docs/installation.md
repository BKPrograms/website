# Installation

## Requirements

Note: Only the core requirements are listed

### Frontend

- Node >= v8.0.0
- Yarn >= v1.22.4
- Vue >= v2.6.12
- Nuxt >= v2.14.7
- Tailwind >= v3.0.0
- Vuetify >= v2.0.0

### Backend

- Firebase >= v7.19.1

## Installation

```
git clone https://github.com/utm-cssc/website.git
cd website
yarn install
yarn run dev
```
