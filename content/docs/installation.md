# Installation

This is the guide to install the CSSC repository on your local machine and to
get started on contributing!

## Requirements

Note: Only the core requirements are listed

### Frontend

- Node >= v8.0.0
- Yarn >= v1.22.4
- Vue >= v2.6.12
- Nuxt >= v2.14.7
- Bootstrap >= v4.5.2
- Vuetify >= v2.0.0

### Backend

- Firebase >= v7.19.1

## Installation

```
git clone https://github.com/utm-cssc/website.git
cd website
yarn install
yarn run dev
```
